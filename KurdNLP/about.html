Kurdish is an Indo-European language spoken by 20 million speakers, inhabitants of Kurdish regions of Turkey, Iran, Iraq and, Syria. Despite the considerable number of speakers who also use Kurdish on their electronic devices in their everyday lives, Kurdish is rarely technically supported online — from a simple spell checker plug-in to Kurdish interfaces— due to the lack of language processing tools.

Previous works

The following table presents the major existing open-source resources and processing tools for Kurdish. This table only provides those resources and processing tools which are supported by a scientific research.

Existing resources  Existing processing tools
 Pewan: The Kurdish Text Corpus / Test Collection (KLPP group)  KurdITGroup keyboards—Arabic and Latin layouts (KurdITGroup community)
 KurdNet, the Kurdish WordNet (KLPP group)  Jedar: a new rule-based stemmer which uses a list of Kurdish suffixes (KLPP group)
 SoraLex, a morphological lexicon for Sorani Kurdish (Alexina project)  Stemmer, tokenizer, Kurdish proper names recognizer and other tools (Hassani, 2017)
 KurLex, a morphological lexicon for Kurmanji Kurdish (Alexina project)  Wergor transliterator (Sina Ahmadi)
